
# sdlRaytracer

first c++ project

raytracer made only using sdl and pure software/CPU rendering no GPU

raytracer features:
- specular, diffuse reflections
- point, directional, ambient lighting
- multithreaded rendering
- fps controls/viewing

renders 15 fps on average
